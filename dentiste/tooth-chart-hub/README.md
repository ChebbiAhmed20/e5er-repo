## Virela - Dental Practice Management

Virela is a modern web application for managing dental practices. It provides tools for managing patients, appointments, billing, referrals, and detailed dental charts in a single, secure workspace.

### Tech stack

- Vite + React + TypeScript
- Tailwind CSS + shadcn/ui
- Supabase (auth, database, edge functions)

### Local development

```sh
cd tooth-chart-hub
npm install
npm run dev
```

The app will start on the port configured in `vite.config.ts` (by default `http://localhost:8080`).
